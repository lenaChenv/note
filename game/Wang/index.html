# 中路教学

# 中单应该捉哪里

  • 有线权：

TIP

看双边那边好捉捉哪边 看对面血线状态 看他们站位靠不靠前 不固定 但是4级前基本对抗路非常好出节奏 这版本也是 大概思路是法刺跟着打野和边路打一片野区和边路 射于辅助抗压

  • 没线权：
    • 跟着对面中单走
    • 在他回来的时候野区丢技能卡他位置 打血量和争夺下一波线权

# 如何快速滚经济

  • 想经济刷得快就要打对面野区
  • 记野区buff刷新时间 1分35秒刷一次
  • 带着队友控对面打野的野区 打野崩了对面整体经济自然落后

# 塔的优先级

  • 中一塔>双边二塔>野区>龙>双边一塔
  • 后期边路高地>中高

# 中一一直推不掉咋办

  • 把边路外塔全推了 法师射手辅助在正面给压力
  • 让队友（刺客 打野 法刺）进野区 在一塔后面给压力 对面还敢守就直接越塔
