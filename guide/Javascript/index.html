# 值类型

Javascript中有五种值类型，分别是：

  • undefined
  • number
  • string
  • boolean
  • Symbol
let a                   // undefined
const n = 100           // number
const s = 'abc'         // string
const b = true          // boolean
const s = Symbol(‘s’)   // Symbol
